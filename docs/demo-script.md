# 10-Minute Demo Script

## 0:00 Intro

Introduce the project: a smart platform connecting candidates with recruiters for internships and freelance missions.

## 1:00 Architecture

Show the three applications: Spring Boot backend, Angular backoffice, Expo mobile app. Explain Firebase Authentication and MongoDB.

## 2:00 Candidate Mobile Login/Register

Open mobile app. Register or login as a candidate. Explain Firebase ID token and backend sync.

## 3:00 Browse Offers and Apply

Show published offers, filters, offer details and application form.

## 4:00 Recruiter Creates Company and Offer

Login as recruiter. Create company and prepare an offer in draft status.

## 5:00 Admin Validates Company in Backoffice

Open Angular backoffice as admin. Validate the recruiter company.

## 6:00 Recruiter Publishes Offer

Return to recruiter view. Publish the offer after company approval.

## 7:00 Candidate Favorites and Applies

Return to candidate. Favorite the published offer and apply with a message.

## 8:00 Recruiter Changes Application Status

Recruiter opens applications, reviews candidate message and changes status to `INTERVIEW`.

## 9:00 Premium and AI Feature

Candidate upgrades to premium with simulated payment and runs CV analysis or offer recommendation.

## 10:00 Conclusion

Show admin dashboard metrics and summarize security, deployment and extensibility.

# Interlance — Presentation Plan

## Slide 1: Title

Interlance, team members and academic context.

## Slide 2: Problem

Fragmented internship/freelance search and manual recruiter filtering.

## Slide 3: Objectives

Centralize offers, applications, matching, admin validation and analytics.

## Slide 4: Architecture

Mobile app, backoffice, backend, MongoDB and Firebase flow.

## Slide 5: Database

Main MongoDB collections and relationships by ids.

## Slide 6: Backend

Spring Boot modules, security, Swagger, tests, Docker and Kubernetes.

## Slide 7: Mobile App

Candidate and recruiter workflows in Expo React Native.

## Slide 8: Backoffice

Admin/recruiter Angular PWA features.

## Slide 9: Security

Firebase token verification, local user sync and role-based access control.

## Slide 10: Deployment

Docker Compose and Kubernetes/Minikube packaging.

## Slide 11: Demo

Live scenario from registration to application status and AI feature.

## Slide 12: Conclusion

Results, limitations, future improvements: real payments, production AI, FCM registration and cloud deployment.
